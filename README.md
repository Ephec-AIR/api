## AIR
### REST api for AIR. 

### Usage
```
git clone https://github.com/Ephec-AIR/api.git
```

```
npm run start
```

> run tests
```
npm run test
```

> production
```
npm run prod
```

> inject fake data
```
node seed-node.js
```
