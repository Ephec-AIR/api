## AIR
### REST api for AIR. 

### Usage
```
git clone https://github.com/Ephec-AIR/api.git
```

```
npm run start
```

> run tests
```
npm run test
```

> production
```
npm run prod
```

> inject fake data
```
node seed.js seed
```

> inject fake data + sync test user (need the server to be started)
```
node seed.js --sync
```
